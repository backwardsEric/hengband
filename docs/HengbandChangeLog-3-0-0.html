Changes

  1. February 7, 2022. 3.0.0 Alpha52 incorporates all of the changes in Hengband's preliminary 3.0.0 versions up to 3.0.0Alpha52. The Mac interface is essentially that of the 2.2.1.7g release with adaptations to work with the 3.0.0 code base. It does use a different directory, ~/Documents/Hengband-3.0.0, than 2.2.1.7g for storing save files, character dumps, and other game data. It also requires a more recent version of macOS, 10.13. The executable was compiled on macOS 11.6.2.
